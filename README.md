## sparkSQL

### Project: Running a Spark Cluster on Jupyter Notebook to explore stock prices data with Spark SQL.

Guided by: Coursera Project Network

### Steps
1. Creating a Spark Context on notebook.

2. Loading three stocks data files.

3. Exploring and Querying data using PySpark Dataframe.
